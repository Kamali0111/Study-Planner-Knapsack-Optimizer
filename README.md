# Study-Planner-Knapsack-Optimizer

## Overview

This project is a web-based application built using HTML, CSS, and JavaScript. It helps users create an optimal study schedule using the 0/1 Knapsack Algorithm from Dynamic Programming.

---

## Description

The Study Planner allows users to input subjects along with required study hours, priority, and difficulty levels. Based on the available study time, the application calculates the most efficient set of subjects to study in order to maximize overall benefit.

The core logic uses Dynamic Programming to solve the knapsack problem, ensuring that users get the best possible schedule within their time constraints.

---

## Features

* **Feature 1: Subject Management**
  Add, view, and remove subjects with hours, priority, difficulty, and optional deadlines.

* **Feature 2: Optimal Schedule Generation**
  Uses the 0/1 Knapsack Algorithm to select the best combination of subjects within the available time.

* **Feature 3: Dynamic UI and Visualization**
  Displays benefit scores, selected subjects, skipped subjects, and overall efficiency in a clean interface.
